# 🎭 Imposter Тоглоом - Монгол хувилбар (Online Multiplayer)

## 📖 Танилцуулга
Энэ бол 5 тоглогчтой, Монгол хэл дээр хийгдсэн **online multiplayer** Imposter тоглоом юм. Жинхэнэ хүмүүс хоорондоо холбогдож, чатлаж, хамтран тоглоно.

## 🚀 Онцлог шинж чанарууд
- **🌐 Real-time Multiplayer**: Socket.IO ашиглан шууд холболт
- **💬 Real-time Chat**: Тоглогчид хоорондоо бодит цагт чатладаг
- **🏠 Room System**: Өрөө үүсгэж, найзуудаа урих боломж
- **📝 Mongolian Topics**: Монгол сэдэв дээр хэлэлцүүлэг хийнэ
- **🗳️ Live Voting**: Бүх тоглогчид хамт санал өгдөг
- **📱 Responsive**: Утас, компьютер дээр ажилладаг

## 🛠️ Суулгах заавар

### 1. Backend суулгах
```bash
# Өмнө нь Node.js суулгасан байх шаардлагатай
npm install
```

### 2. Сервер ажиллуулах
```bash
npm start
# эсвэл
node server.js
```

Сервер ажиллаж эхлэхэд:
```
🚀 Монгол Imposter тоглоом эхэллээ!
🌐 Сервер: http://localhost:3000
📱 Хөтчөөр нээж болно: http://localhost:3000
```

### 3. Тоглох
1. Веб хөтчөөр `http://localhost:3000` хаягыг нээх
2. Өөрийн нэрийг оруулах
3. Шинэ өрөө үүсгэх эсвэл байгаа өрөөнд орох
4. Найзууддаа өрөөний кодыг илгээж тоглох

## 🎮 Тоглох заавар

### 🏠 Өрөөний систем
- **Шинэ өрөө үүсгэх**: 6 оронтой код автоматаар үүснэ
- **Орох**: Өрөөний кодоор орох боломжтой
- **Хамгийн ихдээ**: 5 тоглогч
- **Хамгийн багадаа**: 3 тоглогч (тоглоом эхлүүлэх)

### 🎮 Тоглоомын үе шатууд

#### 1. Хүлээлгээ үе
- Бусад тоглогчид ирэхийг хүлээнэ
- Өрөөний кодыг найзууддаа хуваалцах

#### 2. Тоглоом эхлэл
- Санамсаргүй сэдэв сонгогдоно
- Нэг тоглогч Imposter болно
- Бүх тоглогчид өөрийн дүрийг харна

#### 3. Хэлэлцүүлгийн үе
- **Сэдэв жишээ**: "Хаан сүргийн тухай ярицаа"
- Тоглогчид сэдвийн талаар ярьж, нэг нэгнээ танилцах
- Real-time чат ашиглах
- Imposter-г илрүүлэх оролдлого хийх

#### 4. Санал хураалт
- "Санал хураалт эхлүүлэх" товч дарах
- Тоглогчид Imposter гэж үзсэн хүнээ сонгох
- Бүх санал ирсэний дараа үр дүн гарна

#### 5. Үр дүн
- Хэн ялсан болох харагдана
- Бүх тоглогчийн жинхэнэ дүрүүд ил гарна
- Дахин тоглох боломжтой

## 🌐 Технологи

### Backend:
- **Node.js** - Server runtime
- **Express.js** - Web framework
- **Socket.IO** - Real-time communication
- **UUID** - Room ID generation

### Frontend:
- **HTML5** - Structure
- **CSS3** - Styling with animations
- **JavaScript (ES6+)** - Game logic
- **Socket.IO Client** - Real-time connection
